#### 📥 Entrada:

- A entrada contém dois valores de ponto flutuante de dupla precisão (`double`), correspondentes às notas de um aluno.

*Exemplo de entrada:*
```
5.0
7.1
```

---

<!----------------------------------------------------------------------------------------->

#### 🧠 Lógica:

- Ler as duas notas de ponto flutuante `A` e `B`.
- Calcular a média ponderada sabendo que a nota `A` tem peso **3.5** e a nota `B` tem peso **7.5**. A soma dos pesos é **11.0**:
  $$\text{MEDIA} = \frac{(A \times 3.5) + (B \times 7.5)}{11.0}$$
- Imprimir o resultado com a mensagem `MEDIA = ` e o valor formatado com **5 casas decimais** após o ponto, seguido de uma quebra de linha `\n`.

---

<!----------------------------------------------------------------------------------------->

#### 📤 Saída:

- Imprima a mensagem `MEDIA = ` seguida pelo valor da média do aluno com 5 casas decimais e uma quebra de linha.

*Exemplo de saída:*
```
MEDIA = 6.43182
```

#### 📥 Input:

- The input contains two double-precision floating-point values representing a student's grades.

*Example Input:*
```
5.0
7.1
```

---

<!----------------------------------------------------------------------------------------->

#### 🧠 Logic:

- Read two floating-point numbers `A` and `B`.
- Calculate the weighted average considering grade `A` has a weight of **3.5** and grade `B` has a weight of **7.5**. The sum of weights is **11.0**:
  $$\text{MEDIA} = \frac{(A \times 3.5) + (B \times 7.5)}{11.0}$$
- Print the message `MEDIA = ` followed by the result formatted to **5 decimal places** and an end-of-line break `\n`.

---

<!----------------------------------------------------------------------------------------->

#### 📤 Output:

- Print the message `MEDIA = ` followed by the student's average grade formatted with 5 decimal places and a newline.

*Example Output:*
```
MEDIA = 6.43182
```
