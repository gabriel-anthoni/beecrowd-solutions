#### 📥 Entrada:

- A entrada contém dois valores inteiros (número do funcionário e horas trabalhadas) e um valor de ponto flutuante (valor recebido por hora).

*Exemplo de entrada:*
```
25
100
5.50
```

---

<!----------------------------------------------------------------------------------------->

#### 🧠 Lógica:

- Leia o número do funcionário e as horas trabalhadas usando a função `input()` convertida para `int()`.
- Leia o valor por hora usando a função `input()` convertida para `float()`.
- Calcule o salário multiplicando as horas trabalhadas pelo valor por hora:

$$
SALARY = hoursWorked \cdot salaryPerHour
$$

- utilize f-string para formatar a saída com quebra de linha e 2 casas decimais.

---

<!----------------------------------------------------------------------------------------->

#### 📤 Saída:

- O programa deve exibir o número do funcionário e o seu salário com exatamente duas casas decimais.

*Exemplo de saída:*
```
NUMBER = 25
SALARY = U$ 550.00
```

#### 📥 Input:

- The input contains two integer values (employee's number and worked hours) and one floating-point value (amount received per hour).

*Example Input:*
```
25
100
5.50
```

---

<!----------------------------------------------------------------------------------------->

#### 🧠 Logic:

- Read the employee's number and worked hours using the `input()` function converted to `int()`.
- Read the hourly wage using the `input()` function converted to `float()`.
- Calculate the salary by multiplying the worked hours by the hourly rate:

$$
SALARY = hoursWorked \cdot salaryPerHour
$$

- use f-string to format the output with a newline and 2 decimal places.

---

<!----------------------------------------------------------------------------------------->

#### 📤 Output:

- The program must display the employee's number and their salary with exactly two decimal places.

*Example Output:*
```
NUMBER = 25
SALARY = U$ 550.00
```
