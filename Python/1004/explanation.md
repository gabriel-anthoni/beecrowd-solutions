#### 📥 Entrada:

- A entrada contém dois valores inteiros.

*Exemplo de entrada:*
```
3
9
```

---

<!----------------------------------------------------------------------------------------->

#### 🧠 Lógica:

- Leia os dois valores inteiros usando a função `input()` convertida para `int()`.
- Calcule o produto entre as duas variáveis:

$$
PROD = A \cdot B
$$

- utilize f-string para formatar a saida.

---

<!----------------------------------------------------------------------------------------->

#### 📤 Saída:

- O programa deve exibir o texto `PROD = ` seguido pelo valor correspondente ao produto dos dois números.

*Exemplo de saída:*
```
PROD = 27
```
