#### 📥 Entrada:

*Exemplo de entrada:*
```

```

<!----------------------------------------------------------------------------------------->

#### 🧠 Lógica:

- Leia os três valores inteiros na mesma linha usando `input().split()`.
- Converta a lista de strings obtida para uma lista de inteiros utilizando list comprehension.
- Utilize a função embutida `max()` do Python para encontrar o maior valor entre os números de forma direta e eficiente.
- Exiba o maior valor encontrado seguido da mensagem "eh o maior".
